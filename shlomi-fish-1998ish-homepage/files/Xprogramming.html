Progrmming in X Windows

Well, I want to write my own guide on how to program in Xlib, X Intinstincts, etc., but meanwhile here's a list of useful resources on the various ways to program in X.

Xlib Programming

  • X Window Guide - an on-line tutorial about programming with Xlib. Also covers the X toolkit functions.
  • The Xlib Programming Manual - the official and detailed document from the X Consortium.
  • Xlib Tutorial - a short (and uncompleted) tutorial on Xlib programming.
  • X Windows Version 11.5 - A Concise Description - a short, and hardly elaborate course on the subject.

X Intrinsticts (Xt) Programming

  • X Window Guide - this on-line course also covers the Xlib function.
  • X11 Programming and the Athena Widget Set Course - an HTML transcript of a course that covers the Xt functions library.

Tcl/Tk

  • Tcl/Tk Huge Resource List - Practically everything you'll ever need to learn and program with Tcl/Tk.
  • Perl/Tk FAQ - Using the Tk widget kit with the Perl language. Perl, is a more sensible language than Tcl, IMO, but Perl/Tk is not installed on most systems.
